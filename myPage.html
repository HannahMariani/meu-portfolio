HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto") é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML.

CSS

Cascading Style Sheets (abreviado CSS) é um mecanismo para adicionar estilos (cores, fontes, espaçamento, etc.) a uma página web,aplicado diretamente nas tags HTML ou ficar contido dentro das tags. Também é possível, adicionar estilos adicionando um link para um arquivo CSS que contém os estilos. Assim, quando se quiser alterar a aparência dos documentos vinculados a este arquivo CSS, basta modifica-lo.

Bootstrap

Bootstrap (anteriormente Twitter Bootstrap ) é um framework CSS gratuito e de código aberto direcionado ao desenvolvimento web front-end responsivo e mobile-first . Ele contém modelos de design baseados em HTML , CSS e (opcionalmente) JavaScript para tipografia , formulários , botões , navegação e outros componentes de interface.
